# Domain-Specific AI Copilot

Participants are encouraged to:

- Select a well-defined industry use case
- Incorporate domain terminology, constraints, and decision patterns
- Build Agent Skills or custom skills aligned to the workflow
- Demonstrate a complete decision-support experience

## Judging Focus

### Real-World Relevance

- Is the copilot grounded in authentic industry workflows and business requirements?

### Technical Execution

- Does the solution use domain-appropriate data, guardrails, and effective CoCo CLI orchestration?

### Solution Completeness

- Can the copilot handle the full workflow from context gathering to recommendation and user action?
